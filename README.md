Interactive 3D Portfolio Website 🚀
A modern, high-performance personal portfolio featuring immersive 3D interactions and physics-based animations.
## 🛠️ Tech Stack
* **Frontend**: React, TypeScript, TailwindCSS/Vanilla CSS, Vite
* **3D & Physics**: Three.js, React Three Fiber, Rapier Physics (`@react-three/rapier`)
* **Animations**: GSAP (GreenSock Animation Platform)
## ✨ Key Features
* **Interactive 3D Avatar**: Real-time cursor and touch-motion tracking.
* **Physics Playground**: Interactive, gravity-bound tech stack bubbles with rigid-body collisions.
* **Custom Cursor**: GSAP-powered magnet cursor that snaps to interactive elements.
* **Liquid Animations**: Smooth timeline-driven scroll transitions.
